# Mountain Waterman PWA v1

## Phone setup

### Easiest deployment
1. Upload this folder to GitHub.
2. Connect the repo to Vercel.
3. Deploy.
4. Open the Vercel URL on your phone.
5. Android Chrome: tap menu → Add to Home screen.

## Local setup
npm install
npm run dev

## ChatGPT connection
This app is not automatically connected to ChatGPT yet. Your best workflow for now:
1. Log workouts in the app.
2. Export/copy notes or screenshots.
3. Send them to ChatGPT.
4. ChatGPT can adjust the program and generate a new app version.

A true ChatGPT-connected app would require an OpenAI API key and a small backend/server so your key is not exposed in the browser.
